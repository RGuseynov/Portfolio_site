Prédiction sur le cours d'actifs boursiers

Ce projet a deux objectifs majeurs :

  • Développer des modèles prédictives de façon hybride, en local et sur le cloud.
  • Déployer une solution complète sur le cloud, de la récupération des données jusqu'à l'utilisation final des modèles entraînés.

Données sources :

  • Kaggle : historique de prix du bitcoin à la granularité horaire depuis 2012.
  • Alpha Vantage : API où j'ai récupérer l'historique de prix des 500 entreprises américaines les mieux côtés (S&P500) à la granularité horaire (avec seulement 2 ans d'historique par contre).

Étapes du projet :

  • Développement d'une base de données Azure SQL Serveur et mise en place d'un Azure Key Vault pour stocker les crédentials de manière sécurisé dans le contexte d'Azure.
  • Mise en place de process ETL avec Azure Data Factory pour remplir la base de données depuis l'API Alpha Vantage.
  • Configuration de l'espace de travail Azure Machine Learning (donnée source/compute/environnement python).
  • Traitement et features engineering des données (création de centaines de nouvelles features).
  • Développement de modèles de machine learning de manière hybride (local et cloud).
  • Évaluation et backtesting des modèles.
  • Développement et deploiement d'une Azure Function qui récupère les denièrs prix et effectue des prédictions toutes les heures et d'une page Web pour afficher les dernières prédictions.
  • Rédaction du rapport.

Résultats :

Après avoir globalement remplis mes objectifs hormis celui de devenir le nouveau loup de Wall Street, j'ai mis le projet en stand-by et les ressources Azure sont décommissionnées.
