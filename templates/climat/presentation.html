Etude sur le climat européen

Le but de ce projet est de faire une étude statistique sur le climat européen et plus particulièrement français à partir de données météorologiques et de faire une analyse visuel.

Le traitement des données et le machine learning sont effectués en Python avec Pandas et Scikit-learn.
L'analyse visuel et le compte rendu est effectué avec Power BI.

Données :

Données mondiaux journalier provenant de stations météorologiques

La source met à disposition des données depuis 1929 avec les données les plus complètes à partir de 1973. Dans le cadre de ce projet, je me limiterai aux 20 dernières années.

Etapes du projet :

  • Traitement des données en vue de la création du rapport et pour du machine learning
  • Clusterisation des données pour déterminer les différents climats en France
  • Modélisation d'un rapport visuel avec Power BI
